cass[db]

About

cass[db] is a software tool intended to handle the administration of Cassandra.

cass[db] supports a wide range of operations including managing keyspaces, column families, and columns via the user interface, while you still have the ability to directly execute any CQL statement.

Features

  • Connection to remote and local servers
  • User Authentication
  • Keyspaces
    • List
    • Add keyspace
    • Delete keyspace
    • Custom CQL execution
  • Column Families (CF)
    • List
    • Add CF
    • Delete CF
    • Custom CQL execution
  • Column Family
    • Browse data
      • View row
      • Add row
      • Delete row
      • Edit row
    • Schema
      • View CF schema
      • Add column
      • Delete column
      • Edit column
    • Custom CQL execution
  • Error Handling
    • Connection
    • User Authentication
    • UI Operations
    • CQL execution
